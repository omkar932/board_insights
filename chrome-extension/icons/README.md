# Icon Placeholders

The icons directory should contain:

- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Temporary Solution

For development, you can:

1. Use any PNG images renamed to these filenames
2. Generate icons using an online tool
3. Create simple colored squares as placeholders

## Recommended Design

- Color scheme: Purple gradient (#667eea to #764ba2)
- Icon: Simple chart/analytics symbol
- Style: Modern, flat design

## Generate Icons

You can use tools like:

- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- Or design in Figma/Sketch and export
